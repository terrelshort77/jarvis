## 🔍 Troubleshooting

### 🛑 The app does not open

- Download the file again
- Make sure the file finished downloading
- Right-click the file and choose Open
- Check that Windows did not block the app

### 🎤 The clap does not trigger anything

- Check that the mic is selected in Windows
- Turn up microphone input
- Move closer to the mic
- Reduce room noise
- Try a stronger clap

### 🌐 It opens the wrong app

- Open the app settings if they are available
- Check the app launch links
- Make sure the target apps are installed on your PC
- Confirm your browser can open YouTube links

### 🪟 Windows asks for permission

- Choose Yes if you trust the source
- Run the app as a normal user first
- If needed, move the file out of Downloads and try again
